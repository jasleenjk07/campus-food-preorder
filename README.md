# Campus Food Pre-Ordering Platform

A web-based application designed to reduce food ordering rush in university canteens
by allowing students to pre-order food for specific lecture break time slots.

## Problem
During short lecture breaks, students face long queues in campus canteens,
leading to wasted time and missed meals.

## Solution
This platform enables students to pre-order food in advance and pick it up
during breaks when the food is already prepared.

## Target Users
- University & college students
- Campus canteen vendors

## MVP Features
### Student
- Login
- View canteen menu
- Select break time slot
- Place pre-order
- View order status

### Vendor
- Login
- View upcoming orders by time slot
- Mark orders as prepared

## Tech Stack (Planned)
- Frontend: React
- Backend: Python (FastAPI)
- Database: PostgreSQL
